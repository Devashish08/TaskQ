TaskQ Demo

A Golang Job Queue System with PostgreSQL & Redis

This is a live demonstration of a backend job queue system. The backend is built with Go and uses PostgreSQL for persistent job storage and Redis as a fast, in-memory message broker. This system allows applications to offload long-running or deferrable tasks to be processed asynchronously by background workers.
